# 什么是CC
CC是coding-componse的简称，它是一个提供前后端领域化微服务的组件化开发的组件框架。  

## CC介绍

CC框架分为五部分： 
* coding-cli 脚手架指令
* 前端framework框架
* 前端UI组件
* 前端界面组件
* 后端业务组件

## 依赖环境

* java 17+
* nodejs 16+
* docker-compose 2.x

## 依赖技术

### 前端技术

前端主要使用 react+typescript开发，UI采用antd框架。
* react
* typescript
* ant design pro 
* umi@max
* pro-components
* antd 

### 后端技术
后端主要采用 java+springboot开发，采用领域驱动设计思想。
* springboot
* springboot security
* springboot jpa
* springboot-framework
